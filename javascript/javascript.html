1. Aprendendo JavaScript

1.1. Testando comandos

1.1.1. o que o JavaScript pode fazer?

  • JavaScript é uma linguagem de programação da web.
  • Ele pode calcular, manipular e validar dados.
  • Ele pode atualizar e alterar tanto HTML quanto CSS
  • Ela pode melhorar a dinâmica e interatividade de sua página através de possibilidade de se realizar cálculos, verificar o conteúdo de formulários, escrever jogos interativos adicionar efeitos especiais, customizar a seleção de graficos, criar senhas de segurança e muito mais

1.1.2

  • A tag "script" é usada para incorporar um script do lado do cliente (JavaScript)
  • O "script" elemento contém instruções ou aponta para um arquivo de script externo através do atributo src.
  • Usos mais comuns do JavaScript incluem manpulação de imagens, validação de formulários e alterações dinâmicas de conteúdo.
